## Установка и использование:
### Linux install:
* install dependencies

```make install```

* build

```make publish```

* install

```sudo npm link```

### Usage

```gendiff ---format [type] pathfirstconfig pathsecondconfig```

### fotmats type
* tree
* plain
* json
### supported files
* json
* yaml
* ini
