## Установка и использование:
### Linux install:
* install dependencies

```make install```

* create pacage

```make publish```

* install pacage

```sudo npm link```

### Usage

```gendiff ---format [type] pathfirstconfig pathsecondconfig```

### fotmats type
* tree
* plain
* json
### supported files
* json
* yaml
* json
